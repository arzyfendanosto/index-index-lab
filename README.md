# index-index-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放和展示一系列独立的 HTML 页面。这些页面可以是静态展示页、小型工具页、实验性页面或信息归档页，不针对任何特定域名或网站。仓库本身作为页面集合的索引和载体，方便浏览和引用。

## 目录说明

```
index-index-lab/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面共用的静态资源（如样式、脚本、图片）
├── index.html      # 仓库主页 / 页面索引
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名清晰，便于识别。
- `assets/`：存放页面间可能共用的资源文件，减少重复。
- `index.html`：作为仓库的入口页面，可提供页面列表或导航。

## 页面归档说明

所有归档的 HTML 页面均保持独立，不依赖外部服务或动态后端。页面内容以静态方式呈现，可直接通过浏览器打开或部署到任何静态托管平台。

- 页面之间无跨域依赖。
- 每个页面均可独立访问和分享。
- 归档内容以信息展示和功能演示为主，不包含商业或营销性质的内容。

## 维护说明

- 新增页面时，请在 `pages/` 目录下创建对应文件或文件夹，并在 `index.html` 中添加链接。
- 资源文件请优先放置于 `assets/` 目录，避免页面内嵌过多冗余数据。
- 请保持页面命名简洁、语义化，避免使用无意义的数字或乱码。
- 本仓库仅负责页面归档与发布，不保证内容的长期更新或可用性。
